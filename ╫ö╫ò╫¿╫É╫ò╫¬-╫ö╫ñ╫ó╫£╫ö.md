# הסדר היומי — מדריך הפצה

התיקייה הזו מכילה את האפליקציה כ-PWA (Progressive Web App) מלא, מוכן לשלושה שימושים:
אתר באינטרנט, אפליקציה למחשב, ואפליקציית אנדרואיד.

## מה יש בתיקייה
- `index.html` — האפליקציה עצמה (כולל הכל בפנים)
- `manifest.webmanifest` — הגדרות ההתקנה (שם, אייקונים, צבעים)
- `sw.js` — Service Worker (עבודה offline והתקנה)
- `icon-*.png` / `apple-touch-icon.png` — אייקונים

**חשוב:** כל הקבצים צריכים להישאר יחד באותה תיקייה.

---

## שלב 1 — העלאה לאינטרנט (GitHub Pages)

זה הבסיס לכל השאר. בלי זה אי אפשר להתקין את האפליקציה (PWA דורש כתובת https אמיתית).

1. היכנס ל-github.com והתחבר (או צור חשבון חינם)
2. לחץ על "+" בפינה הימנית העליונה → "New repository"
3. תן שם, למשל `hasseder-hayomi`, סמן **Public**, לחץ "Create repository"
4. בדף הריפו החדש לחץ "uploading an existing file"
5. גרור לתוך העמוד את **כל** הקבצים מהתיקייה הזו (index.html, manifest, sw.js, כל האייקונים)
6. לחץ "Commit changes"
7. לך ל-**Settings** (בתפריט העליון של הריפו) → **Pages** (בתפריט הצד)
8. תחת "Branch" בחר `main` ותיקייה `/ (root)`, לחץ **Save**
9. המתן דקה-שתיים. תופיע כתובת כמו:
   `https://username.github.io/hasseder-hayomi/`

זהו — האפליקציה חיה באינטרנט, נגישה מכל מכשיר.

## שלב 2 — התקנה כאפליקציה במחשב

אחרי ששלב 1 עובד:

1. פתח את הכתובת ב-**Chrome** או **Edge** (במחשב)
2. בשורת הכתובת מימין יופיע אייקון התקנה (מסך קטן עם חץ), או:
   תפריט (⋮) → "Install הסדר היומי" / "התקן אפליקציה"
3. לחץ "Install"

עכשיו יש לך אייקון בשולחן העבודה ובתפריט התחל, והאפליקציה נפתחת בחלון נקי בלי סרגל דפדפן — כמו תוכנה רגילה.

## שלב 3 — אפליקציית אנדרואיד (APK אמיתי)

הדרך הכי פשוטה, בלי Android Studio, דרך PWABuilder (כלי חינמי של מיקרוסופט):

1. היכנס ל-**pwabuilder.com**
2. הדבק את כתובת האתר שלך (מהשלב הראשון) ולחץ "Start"
3. הכלי יבדוק את ה-PWA (אמור לקבל ציונים גבוהים) → לחץ "Package for stores"
4. בחר **Android** → "Generate Package"
5. במסך שנפתח, אפשר להשאיר את הכל בברירת מחדל (או לשנות שם החבילה). לחץ "Download"
6. תקבל קובץ ZIP שמכיל:
   - `app-release-signed.apk` — זה הקובץ שמתקינים ישירות על טלפון אנדרואיד
   - הוראות ומפתחות חתימה (שמור אותם אם תרצה לעדכן בעתיד)

### להתקין את ה-APK על הטלפון
- העבר את קובץ ה-APK לטלפון (מייל / כבל / Drive)
- פתח אותו בטלפון. ייתכן שתצטרך לאשר "התקנה ממקורות לא ידועים" (הגדרה חד-פעמית)
- האפליקציה תותקן עם אייקון משלה

### להעלות לחנות Google Play (רשות)
דורש חשבון מפתחים של Google Play (תשלום חד-פעמי של 25$). ה-ZIP מ-PWABuilder כולל גם קובץ `.aab` שמתאים להעלאה לחנות, עם הוראות.

**חלופה עוד יותר פשוטה (בלי APK):** בכרום בטלפון, פתח את הכתובת → תפריט (⋮) → "Add to Home screen" / "הוסף למסך הבית". זה נותן אייקון ופתיחה במסך מלא, מתנהג כמו אפליקציה, בלי צורך ב-APK בכלל.

## עדכונים עתידיים

כשתרצה לעדכן את האפליקציה (עיצוב חדש, פיצ'רים), פשוט העלה מחדש את `index.html` לריפו ב-GitHub.
העדכון יגיע אוטומטית לכל המכשירים ברענון הבא (האפליקציה בנויה כך שהדף הראשי תמיד נטען מהרשת כשהיא זמינה, והמטמון משמש רק במצב offline).

## הערות
- **סנכרון Firebase** עובד בכל שלושת המצבים — הנתונים מסתנכרנים בין כל המכשירים.
- **פיצ'ר ה-AI** (הוספה בדיבור) דורש מפתח Claude API משלך, שמוזן פעם אחת בתוך האפליקציה.
- זכור להגדיר את כלל האבטחה הקבוע של Firestore (כפי שהוסבר קודם), אחרת הסנכרון יפסיק אחרי 30 יום.
